# JS变量、作用域与内存

## 基本类型与引用类型
```
基本类型值指的是简单数据段，而引用类型值则为可能有多个值构成的对象。
在将一个值赋给一个变量时，解析器必须确定它是基本类型还是引用类型。
对于五种基本数据类型，可以操作保存在变量中的实际的值，所以是按值访问的；
而引用类型的值是保存在内存中的对象，js不允许直接访问内存中的位置，即不能直接操作对象的内存空间。
所以在操作对象时，操作的是对象的引用，而非实际的对象。因此，引用类型的值是按引用访问的。
当然，这并不是非常严密，在给对象添加属性时，操作的是实际的对象。
```

## 变量的复制
```
当向一个变量复制基本类型变量时，实际上就类似于Python中的深拷贝。
首先创建一个新值，然后把该值复制到为新变量分配的位置上，结果就是内存中存了两个被复制的值。
当向一个变量复制引用类型变量时，实际上就类似于Python中的浅拷贝。
将存储在变量对象中的值复制一份到为新变量对象分配的位置，不同的是，复制的值是一个指针，指向的是存储在堆内存中的一个对象。
复制结果就是，两个变量引用同一个对象，如果改变其中一个变量，就会影响另一个变量。
```
